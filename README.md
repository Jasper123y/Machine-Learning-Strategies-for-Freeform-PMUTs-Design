# Machine Learning Strategies for Freeform PMUTs Design

We provide the presentation poster of the work **"Machine Learning Strategies for Freeform PMUTs Design"**.

## Presentation Poster

The poster is available in the [`poster/`](poster/) directory of this repository.

To view or download the poster, navigate to the [`poster/`](poster/) directory and open the poster file.

## About

This repository hosts the presentation materials for the research work on machine learning strategies applied to the design of freeform Piezoelectric Micromachined Ultrasonic Transducers (PMUTs).
